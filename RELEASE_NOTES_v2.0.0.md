- **F15 Key** (Default) - Silent, non-intrusive key press

1. **Set Duration**: Choose Infinite for continuous running, or select Timed with a preset/custom duration
2. **Choose Mode**: Select your preferred keep-awake method from the dropdown
3. **Enable Smart Mode**: Check to automatically pause when you're using your PC
4. **Get Notified**: Enable "Notify when stopped" to get an alert when timed session ends
